# glass-jax Documentation Index

This directory is organized into stable guides and result artifacts.

## Guides

- `launch_doc.md`: original glass-jax launch note and integration overview.
- `guides/`: implementation and project-facing guides.
- `rl/`: reinforcement-learning integration notes.
- `seclust/`: SEClust-specific method, protocol, and report notes.

## Results

- `results/same_protocol/raw/`: seed-level same-protocol JSON artifacts.
- `results/same_protocol/aggregated/`: aggregated same-protocol JSON artifacts.
- `results/same_protocol/reports/`: Markdown same-protocol reports.
- `results/same_protocol/checkpoints/`: recoverable checkpoint snapshots from long runs.
- `results/legacy/`: older benchmark reports not generated by the same-protocol harness.
- `results/figures/`: copied result figures.

`experimental_reports/` is retained as a compatibility location because the benchmark harness currently writes there by default. New paper references should prefer `results/`.
